# API REST Server
API REST con express y MongoDB

## Instalación
```node
npm install
```

## Variables de entorno
```bash
APP_PORT="puerto a usar"

DB_CLUSTER="Cluster de BD Mongo"
DB_NAME="Nombre BD Mongo"
DB_USERNAME="Usuario BD Mongo"
DB_PASS="Contraseña BD Mongo"
```

## Inicio de App
```node
npm start

# o

node index.js
```
